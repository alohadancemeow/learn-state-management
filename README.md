# Learn State Management

A collection of Next.js applications demonstrating different state management libraries and patterns in React.

## Projects

### 1. Admin Dashboard with Redux

**Directory:** `admin-dashboard-with-redux`

- **State Manager:** Redux Toolkit (RTK) & RTK Query
- **Features:**
  - User Authentication (Login/Auth slices)
  - Product & User Management (CRUD operations)
  - Shopping Cart & Checkout Flow
  - Asynchronous data fetching with RTK Query

### 2. Movie Discovery with Zustand

**Directory:** `movie-discovery-with-zustand`

- **State Manager:** Zustand
- **Features:**
  - Movie Search & Trending lists (TMDB API)
  - Genre Filtering
  - Favorites Management (Persisted state)
  - Light/Dark Theme toggle

### 3. Weather Comparison with Jotai

**Directory:** `weather-comparision-with-jotai`

- **State Manager:** Jotai
- **Features:**
  - Atomic state management for city selection & weather data
  - Multi-city Weather Comparison
  - Global Weather Highlights (Asia, Europe, etc.)
  - 5-day Forecast Charts
  - Loadable atoms for async state handling
